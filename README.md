📊 Sales Performance Dashboard (Excel)
🔍 Overview

This project presents an interactive Excel dashboard analyzing sales, profit, and customer order behavior across regions, categories, and shipping classes.

📁 Dataset
Sales transactions dataset (orders, products, regions, categories)
Includes fields like Sales, Profit, Discount, Order ID, Category, and Ship Mode

⚙️ Tools Used
Microsoft Excel
Pivot Tables
Data Model (for distinct count)
Slicers & Interactive Filters
Data Visualization (Charts)

📌 Key Features
Dynamic KPI cards (Total Sales, Total Profit, Top Region, Profit Margin)
Region-wise performance analysis (Sales vs Profit)
Category and Sub-category insights
Discount vs Profit relationship analysis
Distinct Order Count using Data Model
Interactive filtering by Region, Category, and Date

📈 Key Insights
West region leads in both sales and profitability
Central region shows low profit margin despite good sales
Technology category is the most profitable
High discounts negatively impact profit margins
Some high-volume products are loss-making

🧠 Key Learning
Importance of distinct count vs total count
Avoiding incorrect aggregation of percentages
Building business-focused dashboards
Translating data into actionable insights
🚀 Conclusion

This dashboard helps stakeholders quickly identify performance trends, profitability issues, and opportunities for optimization.
